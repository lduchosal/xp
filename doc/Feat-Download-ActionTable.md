# Download XP action table

XP module's action can be programme into the module flash memory


## Action table

Human readable 

```
CP20 0 0 > 1 OFF;
CP20 0 0 > 2 OFF;
CP20 0 0 > 3 OFF;
CP20 0 0 > 4 OFF;
CP20 0 1 > 1 ON;
CP20 0 1 > 2 ON;
CP20 0 1 > 3 ON;
CP20 0 1 > 4 ON;
```
